**R2Droid** is a modern, native Android GUI for the [Radare2](https://github.com/radareorg/radare2) reverse engineering framework. Built with Kotlin and Jetpack Compose, it brings powerful binary analysis capabilities to mobile devices with a clean, touch-friendly interface.

[**🇨🇳 中文说明**](README_CN.md)

## ✨ Features

*   **Project Management**: Save, load, and restore analysis sessions. Support for custom project directories.
*   **Hex Viewer**: High-performance, virtualized hex editor with editing capabilities (`wx`, `w`, `wa`).
*   **Disassembly**: Infinite scrolling disassembly view with syntax highlighting, jump arrows, and navigation history.
*   **Decompiler**: View pseudo-code generated by r2's analysis plugins.
*   **Embedded Terminal**: Full-featured terminal environment using `r2pipe` and `termux-view` for executing raw r2 commands.
*   **Analysis Views**:
    *   Binary Overview (Arch, Bits, OS, etc.)
    *   Sections, Symbols, Imports, Relocations
    *   Strings and Functions list
    *   Cross-references (Xrefs)
*   **Customization**: Dark/Light themes, custom fonts, language switching (English/Chinese), and custom `.radare2rc` configuration.

## 🛠️ Build

1.  Clone the repository.
2.  Open in Android Studio (Ladybug or newer recommended).
3.  Ensure JDK 21 is selected as the Gradle JVM.
4.  Build and Run on an Android device (min SDK 24).

> **Note**: The app includes a pre-compiled `radare2` binary and assets which are extracted upon first launch.
